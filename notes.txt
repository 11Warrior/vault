needed tools :
    psql
    postgresql : 17 
    mongodb-tools(mongodump, mongorestore)
    mongosh

    User runs:

        sudo savr backup postgres -t 6h


    Flow:

        1. Immediate backup runs
        2. SystemdScheduler installs:
            savr-postgres.service
            savr-postgres.timer
        3. systemd handles execution

    After 6 hours:

        systemd
        ↓
        ExecStart=/absolute/path/to/savr backup postgres
        ↓
        CLI runs normally
        ↓
        BackupManager runs


        No background Node process.
        Pure OS-level scheduling.
